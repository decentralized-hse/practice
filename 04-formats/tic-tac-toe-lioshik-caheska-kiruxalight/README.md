# Крестики-Нолики на Beagle

Коллаборативная игра в крестики-нолики. В качестве хранилища и хостинга используется система контроля версий [Beagle](https://github.com/gritzko/librdx).

## Зависимости

- Собранный `be` и `be-srv` из [librdx](https://github.com/gritzko/librdx) (см. инструкцию по сборке в README репозитория)
- Python 3

## Быстрый старт

### 1. Инициализация Beagle-репозитория

```bash
be post //tictactoe/app
```

### 2. Коммит файлов в Beagle

```bash
make commit
```

### 3. Запуск Beagle-сервера

```bash
make serve
```

Сервер запустится на порту 8080.

### 4. Запуск прокси (в отдельном терминале)

```bash
make proxy
```

Прокси запустится на порту 9090. Он нужен потому что HTTP-парсер Beagle некорректно обрабатывает заголовки от браузера.

### 5. Открыть в браузере

```
http://localhost:9090/index.html
```

Для игры вдвоём откройте эту ссылку в двух вкладках/браузерах.

## Полезные команды

```bash
# Посмотреть содержимое файла в Beagle
curl http://localhost:8080/game_index.txt

# Посмотреть конкретную игру
curl http://localhost:8080/game_<id>.txt
```
