Этот репозиторий создан для того чтобы написать простое веб приложение на JavaScript, которое реализует игру в крестики и нолики. Игра должна быть коллаборативной. 

Само по себе задание довольно простое. Но все осложняется тем, что нужно использовать кастомную систему контроля версий Beagle. 
Это собственная разработка нашего преподавателя на курсе по децентрализованным системам (на этом курсе нам и задали задания с крестиками и ноликами). 

Тебе обязательно нужно прочитать эти две статьи чтобы понимать что это за система контроля версий
1. https://replicated.wiki/blog/partI.html
2. https://replicated.wiki/blog/partII.html

Поскольку эта система контроля версий имеет HTTP-api, приложение должно быть как-бы "задеплоино" в эту систему контроля версий. То есть предполагается, что человек в браузере просто откроет ссылку в системе beagle на какой-то один файл (например index.html) и у него запуститься приложение.

Важный нюанс: в качестве БД должен использоваться тоже beagle. То есть какие-то состояния игроков и/или игр тоже должны храниться внутри Beagle.

Для того чтобы понять какими командами можно поднимать beagle сервер, и пользоваться им стоит изучить сам репозиторий с проектом https://github.com/gritzko/librdx. Вроде как в нем есть также какая-то библиотека, для того чтобы делать запросы в beagle репозиторий прямо из браузера.
