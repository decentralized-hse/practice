### Система код ревью (lgtm)

Работа представляет из себя небольшое приложение на js + верста HTML/CSS и моделирует работу code-review системы. Через него можно делать `review-request`, в который можно добавить название измененного файла и его diff. После можно добавлять комментарии к rr и добавлять решения по rr о том, надо ли переделать код или все хорошо.

Код приложения и данные полностью лежат в beagle. Для использования необходимо выполнить:
```bash
be post index.html
be-srv 8080
```

Далее перейти на `localhost:8080/index.html`, данные в формате json можно посмотреть по адресу `localhost:8080/data/reviews.json`.
