### Задача 10: "How is Rust doing in the kernel? Give some stats on the progress of the effort."

В рамках данной задачи мы провели небольшое исследование репозитория Linux (https://github.com/torvalds/linux) на предмет Rust кода. Изначально нами была предпринята попытка воспользоваться для получения необходимой статистики GithubAPI, однако данный инструмент не обладает достаточной функциональностью для полноценного получения нужной информации. Поэтому нами было принято решение проанализировать первые версии (релизы/тэги) за последние N лет - в 2021 году оказалось ровно 0 файлов на Rust, поэтому точкой отсчёта был выбран именно этот год.
